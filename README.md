# GreenPal - Professional Lawn Care Services App

React application for booking lawn care services.

## Installation & Run

```bash
npm install
npm run dev
```

Open `http://localhost:3000`

## Build

```bash
npm run build
```

## Routes

- `/` - Landing Page
- `/loading` - Search Loading
- `/no-results` - No Results Found
- `/results` - Search Results

## Tech Stack

React, Vite, Tailwind CSS, React Router DOM
